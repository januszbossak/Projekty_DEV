## 4. Odrzucone wymagania

### Edytowalne tabele do masowego wprowadzania danych

**Status:** Odrzucone (wymaga nowej wyceny)
**Data zgłoszenia:** 2025-08-28

**Zgłoszenie klienta:**
Klient zgłasza, że obecny sposób rejestracji czasu pracy jest nieefektywny i pracochłonny. Oczekuje rozwiązania przypominającego tabelę przestawną w Excelu do szybkiego wprowadzania godzin.

**Proponowane rozwiązanie (odrzucone):**
- Dashboard z dwoma połączonymi raportami
- Raport typu pivot (macierz zlecenia/dni)
- Raport tabelaryczny z formularzem edycji po kliknięciu komórki

**Decyzja (2025-08-28):**
Oczekiwania klienta wykraczają poza zakres prac uzgodniony na etapie analizy. Obecnie zaimplementowane rozwiązanie zostało wcześniej zaakceptowane. Wszelkie zmiany w tym obszarze będą traktowane jako nowe wymaganie i podlegają osobnej wycenie.
