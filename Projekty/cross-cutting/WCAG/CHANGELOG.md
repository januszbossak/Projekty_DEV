## 2025-08-26 - Notatka projektowa - AMODIT UI

**Źródło:** [[../../../Notatki/Gotowe-notatki-archiwum/2025-08-26 Notatka projektowa - AMODIT UI|2025-08-26 AMODIT UI]]

**Kategoria:** #Dostępność #Funkcjonalność

**Kontekst:**
Implementacja dostępności WCAG 2.1 AA w AMODIT. Główna implementacja dla klienta WIM - zobacz szczegóły: [[../../Klienci/WIM/WCAG/CHANGELOG|WIM/WCAG]]

### Funkcjonalności

Zapewnienie zgodności z normami dostępności WCAG 2.1 AA:
- Tryb wysokiego kontrastu
- Tryb uproszczonego widoku sprawy dla czytników ekranowych
- Nawigacja klawiaturą (Tab, Shift+Tab, strzałki)
- Deklaracja zgodności ze standardem WCAG 2.1 AA

### Szczegóły techniczne

- Opcja włączana na poziomie użytkownika
- Standard: WCAG 2.1 AA
